Método de Bisección

¿Como funciona el Metodo de bisección?

El método de bisección es un método numérico para encontrar la raíz de una función en un intervalo dado. Es un método iterativo que se basa en el teorema del valor intermedio. El método de bisección se puede utilizar para funciones continuas y diferentes en un intervalo cerrado [a, b] donde la función cambia de signo.

El método de bisección funciona de la siguiente manera:

1.-Seleccionar un intervalo cerrado [a, b] donde la función cambia de signo.

2.-Calcular el punto medio c del intervalo: c = (a + b)/2

3.-Evaluar la función en el punto medio c: f(c)

4.-Si f(c) es igual a cero, entonces c es la raíz buscada.

5.-Si f(c) tiene el mismo signo que f(a), entonces la raíz está en el intervalo [c, b]. En este caso, se redefine el intervalo [a, b] como [c, b] y se repite el proceso desde el paso 2.

6.-Si f(c) tiene el mismo signo que f(b), entonces la raíz está en el intervalo [a, c]. En este caso, se redefine el intervalo [a, b] como [a, c] y se repite el proceso desde el paso 2.

El método de bisección es un método lento, pero seguro y confiable para encontrar raíces de funciones. Una de las principales desventajas del método de bisección es que puede requerir un gran número de iteraciones para obtener una precisión suficiente.
